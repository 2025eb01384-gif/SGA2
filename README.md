## Questions Included

### Q1 – Shell Scripting
- Detect duplicate files.
- Back up unique files.
- Generate a report.
- Store errors in a separate log file.

### Q2 – Process Management
- Create child processes using `fork()`.
- Monitor child execution.
- Prevent zombie processes.
- Terminate unresponsive child processes using signals.

### Q3 – File Handling Using Linux System Calls
- Create files using `open()`.
- Write records using `write()`.
- Update records using `lseek()`.
- Retrieve records using `read()`.
- Close files using `close()`.

### Q4 – Linux Command Pipeline
- Monitor log files in real time.
- Extract `ERROR` messages.
- Maintain an error report.
- Suppress unnecessary output using `/dev/null`.

### Q5 – vi Editor Recovery
- Evaluate recovery mechanisms in `vi`.
- Explain swap files, undo history, registers, backup files, and auto-recovery.
- Recommend the most reliable recovery strategy.
